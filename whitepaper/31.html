This model provides a safer approach to royalty distribution than similar Dapps
that rely on a third-party API (such as a block explorer for finding wallet
addresses of token holders). Distribution occurs via a handshake where a
token(s) is sent to a smart contract and royalties along with the token(s) is sent
back to the token holder. Thus if this wallet address is not hosted through our
platform we can verify ownership and safely disburse royalties to the rightful
user.
